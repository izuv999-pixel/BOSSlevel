# BOSSlevel
Это простая игра с боссами проверь свои умения и пройди боссов.
Это динамичный аркадный экшен в жанре Boss Rush,
где игроку предстоит сразиться с тремя уникальными боссами,
каждый из которых обладает своими механиками атак и визуальными эффектами.
Игра требует высокой скорости реакции и грамотного выбора стратегии прокачки.
# Вы можете выбрать одного из четырех героев, каждый из которых полностью меняет стиль игры:

 SPEEDSTER: Высокая мобильность и урон, но критически низкое здоровье.

 TANK: Огромный запас HP, позволяющий выдерживать прямые удары, но низкая скорость.

 BERSERK: Машина для убийства с максимальным уроном, сбалансированная по здоровью.

 GHOST: Неуловимый разведчик с самой высокой скоростью перемещения.
